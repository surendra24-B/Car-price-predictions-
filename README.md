# 🚗 Used Car Price Prediction 

A machine learning and data analysis project focused on predicting used car prices based on vehicle features. This project was developed as a key component of my summer internship, where I applied data science methodologies to solve practical business problems.

---

## 🚀 Project Overview
Accurately estimating the market value of a used car is essential for dealerships, buyers, and sellers. This project utilizes Python and machine learning regression techniques to analyze vehicle datasets, identify major price drivers (such as mileage, brand, and age), and build a predictive model.

## 🛠️ Tech Stack & Libraries
- **Language:** Python 3
- **Data Manipulation:** Pandas, NumPy
- **Data Visualization:** Matplotlib, Seaborn
- **Machine Learning:** Scikit-Learn (Linear Regression, Train-Test Split, Evaluation Metrics)
- **Model Serialization:** Pickle (for exporting and saving the trained model pipeline)

## 📊 Internship Workflow & Key Insights
1. **Exploratory Data Analysis (EDA):**
   - Analyzed how brand reputation and vehicle age impact depreciation.
   - Identified and handled outliers in mileage and pricing data to avoid skewed predictions.
2. **Feature Engineering:** Preprocessed categorical variables (like fuel type and transmission) and scaled numerical features for optimal model performance.
3. **Model Training & Serialization:** Trained a predictive regression model and successfully utilized **Pickle** to export the model (`.pkl` / notebook checkpoints) for potential deployment or future inference.

## 📂 Repository Structure
- `Car Price predictions.ipynb`: The primary notebook detailing data cleaning, visual exploration, and baseline modeling.
- `Car Price Prediction Pickle.ipynb`: The notebook demonstrating model training, evaluation, and saving the finalized pipeline via Pickle.
- `Car Price Predictions.html` & `Car Price Prediction Pickle.html`: Browser-ready HTML exports of the notebooks for quick viewing.
